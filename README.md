**JoinAuction** — это мощное, высокопроизводительное решение для создания аукционов, которое превращает скучную торговлю в динамичную экономическую экосистему. Забудьте о лагах и сложных командах — дайте игрокам интерфейс, в котором хочется тратить деньги!

## ✨ Основные особенности

*   🚀 **Zero-Lag GUI:** Весь интерфейс работает на асинхронных запросах (Project Loom / Java 21). Сервер не «чихнет», даже если в аукционе участвуют сотни игроков одновременно.
*   📦 **Custom Items Support:** Полная поддержка предметов из **ItemsAdder**, **Oraxen**, а также любых кастомных NBT-тегов.
*   📊 **Smart Item Names:** Настраиваемое отображение имен предметов в сообщениях через `item-settings` (поддержка MythicMobs и др.).
*   💰 **Offline Economy:** Игроки получают уведомления и прибыль от продаж сразу при входе на сервер.
*   🎨 **Modern Design:** Поддержка HEX-градиентов (`&{#b50000}`) и юникод-символов в каждом меню.
*   🛡️ **Гибкая система прав:** Настройка слотов, групп и времени хранения через встроенный конфиг или LuckPerms.

## 🛠 Команды


| Команда | Описание                                                |
| :--- |:---|
| `/ah` | Открыть главное меню аукциона                           |
| `/ah sell <цена>` | Выставить предмет из руки на общий рынок                |
| `/ah dsell <цена>` | Выставить предмет на личный аукцион                     |
| `/ah <игрок>` | Посмотреть личный магазин конкретного игрока            |
| `/ah search <запрос>` | Поиск предметов по названию (умный фильтр)              |
| `/ah help` | Список всех доступных команд                            |
| `/ja reload` | Перезагрузить конфигурацию плагина (нужны права админа) |

---

## 🔑 Права (Permissions)

*   `auction.admin` — Доступ к `/ja reload`.
*   `auction.serch` — Право на поиск предметов.
*   `auction.player.serch` — Право на просмотр чужих аукционов.
*   `auction.slot.sell.[N]` — Лимит слотов продажи (замени [N] на число).
*   `auction.slot.dsell.[N]` — Лимит слотов личного аукциона.
*   `auction.slot.time.[N]` — Право на время хранения лота (в часах).
*   `auction.group.[name]` — Привязка к кастомной группе из `config.yml`.

## ⚙ Установка

1.  Скачайте **JoinAuction.jar** и поместите его в папку `plugins`.
2.  Убедитесь, что на сервере установлены **Vault** и плагин на экономику (например, EssentialsX).
3.  (Опционально) Установите **PlaceholderAPI** для поддержки плейсхолдеров.
4.  Перезагрузите сервер.
5.  Настройте `lang/ru_ru.yml` под стиль вашего сервера.
